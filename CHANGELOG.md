# Changelog

## [1.0.0] – 2026-06-15

### Added
- AI Chat Sidebar with 6 modes: Chat, Generate, Debug, Explain, Refactor, Tests
- Inline ghost-text completions (Tab to accept)
- Live web search powered by Claude's search tool
- Streaming responses — text appears word-by-word
- Right-click context menu with 5 DevAI actions
- Keyboard shortcuts for all major commands
- Secure API key storage via VS Code SecretStorage
- Auto-detects active file language and context
- Status bar DevAI button
- Dark/light theme support (follows VS Code theme)
- Insert generated code directly into editor
- One-click copy for code blocks
- GitHub Actions workflow for CI and Marketplace publishing
